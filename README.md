# HR_Dashboard_using_Power_BI
This project entails the comprehensive analysis of the company's employee demographics using a Power BI dashboard. It encompasses essential data points such as the female-to-male ratio, gender distribution, marital status, and geographical distribution of employees. The dashboard offers insights into the recruitment sources that have been most effective, shedding light on the company's hiring strategy. Furthermore, it delves into the reasons for employee termination, identifying trends and patterns. This data-driven approach empowers decision-makers with valuable information for strategic workforce planning. It's crucial to note that the company's data used for this project is strictly confidential and cannot be disclosed due to privacy concerns. The Power BI dashboard serves as a visual aid to interpret and comprehend these multifaceted workforce analytics, facilitating informed decisions without compromising data confidentiality.
